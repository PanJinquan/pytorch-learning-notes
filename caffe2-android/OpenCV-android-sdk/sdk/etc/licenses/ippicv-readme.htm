Intel® Integrated Performance Primitives (Intel® IPP) Library for OpenCV

Intel® Integrated Performance Primitives (Intel® IPP) library for OpenCV is a subset of Intel IPP functions for image processing and computer vision.

For detailed descriptions of Intel IPP functions and interfaces refer to the Intel IPP Reference Manual available from the Intel IPP documentation Web page at http://software.intel.com/en-us/articles/intel-integrated-performance-primitives-documentation/. Note that not all functions described in the Reference Manual are included into the Intel IPP library for OpenCV.
